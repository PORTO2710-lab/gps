# 📍 GPS - Servicio de Geolocalización Web

Este proyecto es un localizador de direcciones en tiempo real que utiliza mapas interactivos y servicios de geocodificación abierta.

## 🚀 Demo
Puedes ver el servicio funcionando aquí: [https://porto2710-lab.github.io/gps/](https://porto2710-lab.github.io/gps/)

## ✨ Características
* **Búsqueda Global:** Localiza cualquier dirección, ciudad o país.
* **Interfaz Dark:** Diseño minimalista y moderno optimizado para la visualización de datos.
* **Mapas Interactivos:** Zoom, desplazamiento y marcadores con popups informativos.
* **Ligero:** Carga rápida gracias al uso de Leaflet.js.

## 🛠️ Tecnologías Utilizadas
* **HTML5 / CSS3:** Estructura y diseño responsivo.
* **JavaScript:** Lógica de búsqueda y manipulación del mapa.
* **Leaflet.js:** Librería de código abierto para mapas móviles.
* **Nominatim API:** Motor de búsqueda para geocodificación.
